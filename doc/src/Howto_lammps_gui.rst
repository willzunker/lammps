Using the LAMMPS GUI
====================

Hotkeys
^^^^^^^

Almost all functionality is accessible from the menu or via hotkeys.
The following hotkeys are available (On macOS use the Command key
instead of Ctrl (aka Control)).

.. list-table::
   :header-rows: 1
   :widths: auto

   * - Hotkey
     - Function
     - Hotkey
     - Function
     - Hotkey
     - Function
     - Hotkey
     - Function
   * - Ctrl+N
     - New File
     - Ctrl+Z
     - Undo edit
     - Ctrl+V
     - Paste text
     - Ctrl+Q
     - Quit (Main Window only)
   * - Ctrl+O
     - Open File
     - Ctrl+Shift+Z
     - Redo edit
     - Ctrl+Enter
     - Run LAMMPS
     - Ctrl+W
     - Close (Log and Image Window only)
   * - CTRL+S
     - Save File
     - Ctrl+C
     - Copy text
     - Ctrl+/
     - Stop Active Run
     - Ctrl+P
     - Preferences
   * - Ctrl+Shift+S
     - Save File As
     - Ctrl+X
     - Cut text
     - Ctrl+I
     - Create Snapshot Image
     - Ctrl+Shift+/
     - Quick Help

Further editing keybindings `are documented with the Qt documentation
<https://doc.qt.io/qt-5/qplaintextedit.html#editing-key-bindings>`_.  In
case of conflicts the list above takes precedence.
